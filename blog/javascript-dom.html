Javascript Fundamentals

An analogy to describe JavaScript and its relationship to HTML and CSS.

A good analogy to explain the relationship between JavaScript, HTML and CSS is to liken it to building a house. The HTML is like the plans and structure of the house.
The CSS is like all of the design and decorating (eg. paint colours, furnishings, decor).
And, finally, JavaScript is like all of the plumbing and electricty. It brings it all together and makes everything work, as well as adding ineractivity.

Explain the difference between accessing data from arrays and objects.

Arrays are an ordered list of items, they are indexed lists and you use this to access the information stored.

For example: [apple, orange, banana] - would be indexed like this [0, 2, 2]. You would use these numbers to access the items.

Objects are unordered collections of information and are accessed via names(keys).

Explain what functions are and why they are helpful.

Functions are blocks of code that form a specific task in your code. They are helpful as they can be reused, so you're not writing the same code over and over and cluttering up your files.
